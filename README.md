WIP next-gen dashboard for openHAMS.

## Features
- Both server and client written in ES6+ with native imports transpiled by Babel.
- Hot reload for back-end

## Requirements

- Node 9+
- MongoDB
- Yarn
- Google api keys for authentication

### Dependencies

- Passport
- Webpack 4
- Vue 2
- Vuex
- Babel 6
- Lodash
- Express
- Mongoose
